# vue-dashboard-boilerplate

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run dev
```

### Compiles and minifies for production
```
yarn run build
```

### Run for production
```bash
# must build first
yarn start
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
